# 标准化岗位清单

本文件将 [../../岗位JD.md](../../岗位JD.md) 中的 30 个岗位统一编号为 `JD-01` 到 `JD-30`，用于横向对比、技术栈提炼和学习计划设计。

> 说明：原始 JD 中未明确的信息标为“未明确”；“岗位方向、业务场景、提炼出的技术能力、对应学习任务”为分析结论，不替代原始 JD。

## 标准字段说明

| 字段 | 说明 |
|---|---|
| JD 编号 | 派生文档统一编号 |
| 原始编号 | 在 [../../岗位JD.md](../../岗位JD.md) 中的编号 |
| 原始批次 | 第一批 / 第二批 / 第三批 |
| 岗位方向 | 面向学习和求职的主分类 |
| 业务场景 | JD 中体现的行业或业务落地方向 |
| 核心能力 | 从职责、要求、标签中提炼出的能力 |
| 对应学习任务 | 后续学习计划可展开的实践任务 |
