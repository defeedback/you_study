# 面试专项模块

面向 JD 的面试准备。与 [teaching.md](teaching.md) 模板里的「面试会怎么问」区别：那是每节课的面试钩子，本模块是系统化面试准备。

## 高频题整理

从 [岗位JD.md](../../岗位JD.md) 提炼高频技术点，为每个点准备：

- **面试会怎么问**：常见问法变体。
- **标准回答思路**：第一性原理推导 + 关键取舍。
- **容易踩坑**：常见错误答案、过时认知。

## 示例（RAG）

- 问：为什么 RAG 不用 Fine-tuning？→ 知识频繁变更、成本、可解释性。
- 问：RAG 检索效果差怎么排查？→ chunk 大小、Embedding 质量、是否缺 Rerank、query 改写。
- 问：向量库怎么选？→ 规模、是否要混合检索、运维成本。

## 模拟面试

用户要求模拟面试时，扮演面试官追问"为什么"，逼用户表达第一性原理与取舍，而非背 API。
