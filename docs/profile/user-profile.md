# 用户画像：AI Agent 应用开发学习

## 画像日期

- 建立日期：2026-06-11
- 目标周期：2 个月
- 每日学习时间：约 1 小时

## 目标方向

用户当前目标方向是 **AI Agent 应用开发**。

根据 [../jd-analysis/02-role-taxonomy.md](../jd-analysis/02-role-taxonomy.md) 和 [../jd-analysis/03-tech-stack-matrix.md](../jd-analysis/03-tech-stack-matrix.md)，该方向的核心要求包括：

- Python 与后端 API 工程
- LLM API 调用
- Prompt Engineering
- Function Calling / Tool Calling
- RAG 知识库系统
- Agent 任务规划、记忆管理、工具调用、多轮对话
- LangChain / LangGraph / LlamaIndex 等框架
- FastAPI、数据库、Docker、日志、评测等工程化能力
- 业务场景理解与项目落地表达

## 当前基础

### 编程能力

- 主修语言：Python
- 能力判断：可以独立完成一个小项目
- 已具备基础：脚本编写、项目实现、基本工程组织能力

### 后端与工程经验

- 使用过 Flask
- 使用过 FastAPI
- 会简单 SQL
- 使用过 Linux 命令
- 使用过 Docker
- 使用过 Git

### AI / LLM 基础

- 调用过大模型 API
- 了解 Prompt Engineering
- 了解工具调用
- 了解 RAG
- 了解 Embedding
- 了解向量数据库

### 项目经验

- 尚未做过完整 Agent 项目
- 尚未做过完整 RAG 项目

## 当前优势

1. **Python 基础匹配岗位主线**
   - 30 份 JD 中 Python 是最高频语言，用户主修 Python，方向匹配度高。

2. **已有 FastAPI / Flask 接触经验**
   - 适合走“AI Agent 应用开发 + 后端工程化”路线，不必从 Web 基础完全开始。

3. **已经了解 LLM / RAG / 工具调用概念**
   - 可以直接进入项目驱动学习，不需要长时间停留在概念科普阶段。

4. **学习目标明确**
   - 目标方向聚焦 AI Agent 应用开发，学习路线可以减少算法训练、模型底层等非主线内容。

## 当前短板

1. **缺少完整 RAG 项目经验**
   - 需要补齐文档解析、分块、Embedding、向量检索、重排、引用来源、评测等完整链路。

2. **缺少完整 Agent 项目经验**
   - 需要补齐工具调用、任务规划、记忆管理、多轮对话、工作流编排等能力。

3. **工程化深度需要加强**
   - SQL、数据库建模、Redis、日志、异常处理、Docker 部署、接口文档、评测体系需要通过项目补齐。

4. **求职项目包装尚未建立**
   - 需要同步沉淀 README、架构图、项目亮点、技术难点、面试问答材料。

## 学习偏好

用户偏好 **效率较高的方式**。

推荐采用：

- 项目驱动优先
- 必要概念随项目补齐
- 每周有可检查产出
- 每个阶段都沉淀文档
- 以求职项目和面试表达倒推学习内容

不建议采用：

- 长时间看理论课但不做项目
- 一开始学习模型训练 / 微调 / 深度学习底层
- 同时铺开太多框架
- 只做 Notebook Demo，不做后端服务和工程化

## 项目兴趣

用户感兴趣方向：

1. 个人知识库
2. 智能客服
3. 代码仓库探索

推荐整合为一个主线项目：

**面向技术学习与代码仓库的智能知识库 Agent**

该项目可以分阶段演进：

1. 个人知识库 RAG
2. 智能客服式多轮问答
3. 工具调用 Agent
4. 代码仓库探索 Agent
5. FastAPI 后端服务
6. Docker 部署、日志、评测和 README 包装

## 推荐学习策略

### 主线策略

用 2 个月做出一个求职级 AI Agent 项目，而不是做多个松散 Demo。

项目主线建议：

> 从个人知识库 RAG 出发，逐步升级为支持工具调用、代码仓库探索、多轮对话和工程化部署的 AI Agent 应用。

### 技术取舍

优先学习：

- Python
- FastAPI
- LLM API
- Prompt Engineering
- Function Calling / Tool Calling
- RAG
- 向量数据库，先用 FAISS / Chroma / PGVector 中一种即可
- LangChain / LangGraph，优先 LangGraph 用于 Agent 工作流
- Docker
- 日志与评测

暂缓学习：

- 模型训练 / 微调
- CUDA / 推理优化
- 多模态 AIGC
- K8s 深入
- 复杂 Multi-Agent 平台

## 学习成功标准

2 个月后应至少产出：

1. 一个可运行的 AI Agent / RAG 项目
2. FastAPI 后端接口
3. 文档上传、解析、检索、问答能力
4. 工具调用能力
5. 简单会话记忆
6. 代码仓库探索能力雏形
7. Docker 启动方式
8. 项目 README
9. 架构图或模块说明
10. 简历项目描述和面试问答材料
